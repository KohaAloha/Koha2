Quelques types de document adaptés à la lecture publique : 
- Documents papiers : Livre, BD, Périodique, Usuel, Article, Document adapté, Fonds local
- Documents multimédias : CD musique, CD-ROM, DVD, VHS
- Documents numériques : Ressource web

(Ces types de documents peuvent être modifiés après l'installation, à partir du module d'administration.)
